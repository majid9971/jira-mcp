# Jira & GitHub MCP Project


This is a basic Node.js project scaffolded to integrate Jira via Model Context Protocol (MCP) for ticket access and GitHub MCP for direct code commits via Copilot agent. All actions are performed through Copilot (AI), not via extensions.

## Features
- Access Jira tickets directly from the project using MCP (no extensions)
- Commit code to GitHub repository using Copilot agent

## Getting Started
1. Install dependencies: `npm install`
2. Start the project: `npm start`

## Integrations
- Jira via MCP (setup required)
- GitHub MCP (setup required)

---

For more details, see the copilot-instructions.md in the .github directory.
